# ビジネスメール・文書作成スキル

ビジネスメール・提案書・お礼状・お断りメール・議事録など、日本語ビジネス文書を作るスキル。

## まず確認すること

文書を作る前に以下を把握する（会話から読み取れる場合は省略可）：

1. **文書の種類** — メール / 提案書 / お礼状 / お断り / 依頼 / 議事録など
2. **相手** — 初対面の取引先 / 既存顧客 / 上司 / 同僚など
3. **目的** — 何を伝え、相手にどう動いてほしいか
4. **トーン** — 丁寧・カジュアル・フォーマル
5. **含めるべき情報** — 日時・金額・具体的な依頼事項など

---

## ビジネスメールの構造

```
件名：【目的が一目でわかる件名】

〇〇株式会社
〇〇部 〇〇様

いつも大変お世話になっております。
〇〇株式会社の〇〇と申します。

[本文：目的→詳細→お願い/次のアクション]

ご確認のほど、どうぞよろしくお願いいたします。

━━━━━━━━━━━━━━━━━━━━
〇〇 〇〇（フルネーム）
〇〇株式会社 〇〇部
TEL: 000-0000-0000
Email: xxx@example.com
━━━━━━━━━━━━━━━━━━━━
```

## シーン別テンプレート

### 初めてのご連絡（新規アプローチ）

```
件名：〇〇のご提案について（〇〇株式会社）

〇〇株式会社
〇〇部 〇〇様

突然のご連絡、大変失礼いたします。
〇〇株式会社の〇〇と申します。

[相手との接点・なぜ連絡したかの一文]

弊社では〇〇というサービスを提供しており、
〇〇様のビジネスにお役に立てるのではないかと思い、
ご連絡させていただきました。

もしよろしければ、一度お時間をいただき、
詳しくご説明させていただけますでしょうか。

ご都合のよいお日にちをいくつかお知らせいただければ
幸いです。

どうぞよろしくお願いいたします。
```

### 打ち合わせのお礼

```
件名：本日はありがとうございました（〇〇の件）

〇〇株式会社
〇〇部 〇〇様

お世話になっております。〇〇株式会社の〇〇です。

本日はお忙しいところ、お時間をいただきありがとうございました。

[打ち合わせで話した内容を1〜2文で要約]

いただいたご意見をもとに、社内で検討を進めてまいります。
改めてご提案の機会をいただけますと幸いです。

引き続き、どうぞよろしくお願いいたします。
```

### 依頼メール

```
件名：〇〇のご依頼について

〇〇様

お世話になっております。〇〇の〇〇です。

[依頼の背景・理由を1文で]

つきましては、以下についてお願いできますでしょうか。

■ 依頼内容
・〇〇
・〇〇

■ ご回答期限
〇月〇日（〇）まで

ご多忙のところ恐れ入りますが、
ご対応のほどよろしくお願いいたします。

ご不明な点がございましたら、お気軽にお問い合わせください。
```

### お断りメール

```
件名：〇〇の件につきまして

〇〇株式会社
〇〇部 〇〇様

お世話になっております。〇〇の〇〇です。

この度は〇〇についてご連絡いただき、誠にありがとうございます。

慎重に検討いたしました結果、
今回は[理由：予算の都合・時期の問題・方向性の違いなど]により、
見送らせていただくことになりました。

せっかくお声がけいただいたにもかかわらず、
このようなご返答となってしまい、大変申し訳ございません。

またの機会に、ぜひよろしくお願いいたします。
引き続きどうぞよろしくお願いいたします。
```

### 資料送付メール

```
件名：〇〇資料のご送付について

〇〇様

お世話になっております。〇〇の〇〇です。

先日ご依頼いただいた〇〇の資料をご送付いたします。

■ 添付資料
・〇〇.pdf（〇〇についての説明資料）
・〇〇.xlsx（見積書）

ご確認いただき、ご不明な点やご質問がございましたら
お気軽にお申し付けください。

どうぞよろしくお願いいたします。
```

## 提案書の構成

```markdown
# 〇〇のご提案

**提案日：** 〇〇年〇月〇日
**提案先：** 〇〇株式会社 〇〇部 御中
**提案者：** 〇〇株式会社 〇〇

---

## 1. 現状の課題

[相手が抱えている課題を相手の言葉で書く。
 「〇〇なので〇〇が大変」という形式が伝わりやすい]

---

## 2. 提案内容

[課題に対して何を提供するかを簡潔に]

### 2-1. 〇〇（サービス/機能名）

- **概要：** 〇〇
- **解決できること：** 〇〇
- **特徴：** 〇〇

---

## 3. 期待される効果

| 項目 | 現状 | 導入後 |
|---|---|---|
| 〇〇 | 〇〇 | 〇〇 |
| 〇〇 | 〇〇 | 〇〇 |

---

## 4. 費用・スケジュール

| 項目 | 内容 | 金額 |
|---|---|---|
| 初期費用 | 〇〇 | 〇〇円 |
| 月額費用 | 〇〇 | 〇〇円/月 |

**想定スケジュール：**
- 〇月：契約・キックオフ
- 〇月：〇〇フェーズ
- 〇月：本番稼働

---

## 5. 次のステップ

ご関心をお持ちいただけましたら、以下のステップをご提案いたします。

1. ご質問・懸念点の確認（〇〇までに）
2. 詳細なご要件のヒアリング
3. 正式なお見積もりのご提出

ご不明な点はお気軽にお問い合わせください。
```

## 文章の品質チェックリスト

文書を仕上げる前に確認：

- [ ] **件名で内容がわかるか** — 「ご連絡」だけでは×
- [ ] **冒頭で誰に・何のメールか明確か**
- [ ] **1メール1トピックか** — 複数の要件がある場合は番号で整理
- [ ] **依頼している場合、期限が明記されているか**
- [ ] **次のアクションが明確か** — 相手に何をしてほしいか
- [ ] **敬語の重複がないか** — 「〜させていただきます」の多用を避ける
- [ ] **読んで3秒で要点が伝わるか**

---

## 敬語・言い回しの使い分け

| 避けたい表現 | 代替表現 |
|---|---|
| 〜させていただきます（多用） | 〜いたします / 〜します |
| ご確認の程よろしくお願いします | ご確認のほど、よろしくお願いいたします |
| 取り急ぎ〜 | 略式ながら〜 / （正式文書には使わない） |
| お世話様です | お世話になっております |
| ご苦労様です（目上に） | お疲れ様です |
| 了解しました（目上に） | 承知いたしました / かしこまりました |
